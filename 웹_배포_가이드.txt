🌐 웹 배포 가이드 - Streamlit Cloud (무료)
=========================================

이제 동료들과 Python 없이도 웹 브라우저로 프로그램을 사용할 수 있습니다!

---

✅ 방법 1: Streamlit Cloud (가장 쉬움, 무료)

1. GitHub에 코드 올리기
   - https://github.com 에서 계정 생성
   - 새 저장소(repository) 만들기
   - exceltoword 폴더의 모든 파일 업로드

2. Streamlit Cloud 연결
   - https://streamlit.io/cloud 접속
   - GitHub로 로그인
   - "Deploy an app" 클릭
   - 저장소 선택
   - Main file: excel_to_word_converter.py 선택
   - "Deploy!" 클릭

3. 완료! 
   - https://your-app-name.streamlit.app 주소가 생성됩니다
   - 이 주소를 동료들에게 공유하세요!

---

✅ 방법 2: 로컬 네트워크 (회사 내부 네트워크용)

터미널에서 실행:
   streamlit run excel_to_word_converter.py --server.address 0.0.0.0

동료들은 브라우저에서 접속:
   http://당신의IP주소:8501
   
IP 주소 확인:
   ifconfig | grep "inet "

---

2. Streamlit Cloud에서 배포
3. 주소 공유

끝!
