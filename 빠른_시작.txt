📄 Excel to Word 변환기 - 빠른 시작 가이드
=============================================

🚀 가장 간단한 사용 방법 (추천)

1. 터미널(Terminal)을 열어주세요
   - Spotlight 검색(Command + Space) → "터미널" 검색

2. 다음 명령어를 입력하세요:
   
   cd ~/Desktop
   cd exceltoword
   python3 excel_to_word_converter.py

3. 자동으로 웹 브라우저가 열리고 프로그램이 시작됩니다!
   (주소: http://localhost:8501)

---

📦 전체 파일 패키지 공유 방법

동료들에게 다음 파일들을 모두 압축해서 보내주세요:

필수 파일:
- excel_to_word_converter.py
- requirements.txt
- 빠른_시작.txt

선택 파일:
- README.md
- 사용법.txt

동료들은 받은 후:
1. 파일을 압축 해제
2. 터미널에서 압축 해제한 폴더로 이동
3. python3 excel_to_word_converter.py 실행

---

⚙️ 필요한 라이브러리 설치

처음 사용하시는 경우, 다음 명령어로 필요한 라이브러리를 설치하세요:

pip3 install -r requirements.txt

---

💡 문제 해결

Q: "command not found: python3" 에러가 나요
A: Python이 설치되어 있지 않습니다.

Q: "ModuleNotFoundError" 에러가 나요  
A: 필요한 라이브러리가 설치되지 않았습니다.
   다음 명령어를 실행하세요:
   pip3 install -r requirements.txt

Q: 프로그램이 실행되지 않아요
A: python3 excel_to_word_converter.py 입력 후 Enter 누르세요
   자동으로 브라우저가 열리면 성공입니다!
